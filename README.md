## Staff Management Application

Staff is a web application that helps to manage employees base.

## Installation

Run these commands in console:

```bash
$ php artisan migrate
```

```bash
$ php artisan db:seed
```

```bash
$ composer dump-autoload
```

```bash
$ php artisan cache:clear
```
